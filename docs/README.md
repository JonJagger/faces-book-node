# faces-book-node
- Look at a person's face and try to remember their name.
- Hover the cursor over their face.
- Their name will appear.
- Were you right?

Implemented using
[Node](https://nodejs.org/en/),
[Express](https://expressjs.com/) and
[Docker](https://www.docker.com/)
